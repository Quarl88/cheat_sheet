Basic Selectors
Target Syntax Description
Type tag {} Styles all instances of selected tag in the HTML code
Class .classname {} Styles instances of the selected class
ID #elementid {} Style element with selected ID (most specific selector)
Attribute [attribute] {} Styles elements with selected attribute
Attribute+Value [attribute*='value'] {} Styles elements with selected property AND value
Pseudo Selectors
Name Syntax Trigger
Focus selector:focus {} User clicks, taps, or otherwise selects element
Hover selector:hover {} Cursor touches or "hovers" over element.
Visited selector:visited {} User has already clicked on the link.
Active selector:active {} The element is "activated" (usually clicked) by the user
nth child selector:nth-child(value) Selects based on position among siblings (odd/even, num)
Media Queries
Name Syntax Description
Screen @media screen {} Targets device displays but not printed
Print @media print {} Allows different styling when printed
Screen Width @media (max-width: 1250px) @media (min-wdith: 600px) Targets styles for different screen sizes (responsive web design essential)
