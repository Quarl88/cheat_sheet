Selectors

Basic Selectors
Target Syntax Description
Type tag { declaration } Styles all instances of selected tag in the HTML code
Class .classname { declaration } Styles instances of the selected class
ID #elementid { declaration } Style element with selected ID (most specific selector)
Attribute [attribute] { declaration } Styles elements with selected attribute
Attribute+Value [attribute*='value'] { declaration } Styles elements with selected property AND value
Pseudo Selectors
Name Syntax Trigger
Focus selector:focus { declaration } User clicks, taps, or otherwise selects element
Hover selector:hover { declaration } Cursor touches or "hovers" over element.
Visited selector:visited { declaration } User has already clicked on the link.
Active selector:active { declaration } The element is "activated" (usually clicked) by the user
